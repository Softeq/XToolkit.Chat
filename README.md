# XToolkit.Chat

Cross-platform implementation of chat for Android and iOS.

## Platforms

- Android
- iOS

## Components

- Xamarin.Android chat UI
- Xamarin.iOS chat UI
- SignalR client
- HTTP client
- Common chat logic
